# 连浊规则

1.（最基本的）两个单词合在一起组成一个词（即合成语），第二个单词的第一个音节，会变浊音（即连浊）。

例：

回転（かいてん）+寿司（すし）

＝回転寿司（かいてんずし）

（不念成浊音的话，就是「回転」和「すし」两个词列出来而已。念成浊音即代表为一个词 。）



2.只有第二个名词的第一个音节是か、さ、た、は行以外的音会变为浊音。（也就是起码这个音能变成浊音）

例：

回転（かいてん）+窓（まど）

＝回転窓（かいてんまど）（无变化）



3.音读（汉字）一般不连浊

例：

運動（うんどう）+会（かい）

＝運動会（うんどうかい）（因为「会（かい）」是音读，所以无变化）



生（せい）+花（か）

＝生花（せいか）

（「花（か）」是音读，所以无变化）



草（くさ）+花（はな）

＝草花（くさばな）

生（い）け+花（はな）

＝生（い）け花（ばな）

（「花（はな）」是训读，所以会有连浊）



同理

花（はな）+火（ひ）

＝花火（はなび）



4.后半段的单词中有浊音的，不连浊。

例：

旅行（りょこう）+かばん

＝旅行（りょこう）かばん（因为「かばん」中的「ば」是浊音，所以不连浊）

（为了顺口考虑）



5.（与3相关的）汉字中也有不连浊的例外（是在日常中非常常用/日本人用很久了的词，约10%）。

例：

旅行（りょこう）+会社（かいしゃ）

＝旅行会社（りょこうがいしゃ）

（虽然「会社」是汉语词，但是是例外）

睡眠（すいみん）+不足（ふそく）

＝睡眠不足（すいみんぶそく）

文庫（ぶんこ）+本（ほん）

＝文庫本（ぶんこぼん）



6.外来词（基本上）不连浊（是使用了很久很久的词，例外很少）。

例：

+

＝

例外：

歌（うた）+かるた

＝歌（うた）がるた（和歌纸片）



（「かっぱ」，意为外套/雨衣，可以写成「合羽」，源葡萄牙语，现已被日语同化，在日本人眼里基本不算外来语）

雨（あめ）+かっぱ

＝雨（あめ）がっぱ







练习：

1.子（こ）+猿（さる）＝子猿（こざる）（小猴子）

2.子+羊（ひつじ）＝子羊（こひつじ）

# 连浊规则AI总结版

## 1. （最基本的规则）组成合成词时，第二个单词第一个音节发生连浊

- 第二个单词的第一个音节变成浊音，表示这两个词合成为一个词。
- 如果不连浊，只是单独列举两个词。

**例：**
 回転（かいてん）+ 寿司（すし）
 ＝ 回転寿司（かいてんずし）

------

## 2. 第二个单词第一个音节**必须是可以变成浊音的假名**

- 即：か、さ、た、は行的音才能连浊。
- 如果第二个词开头不是这几个行的音，就不会连浊。

**例：**
 回転（かいてん）+ 窓（まど）
 ＝ 回転窓（かいてんまど）（无变化）

------

## 3. 音读的词（多为汉字）一般**不连浊**

- 训读（和语读音）可以连浊，音读（汉语音）通常不会连浊。

**例子：**

- 運動（うんどう）+ 会（かい）
   ＝ 運動会（うんどうかい）（无变化）
- 生（せい）+ 花（か）
   ＝ 生花（せいか）（无变化）

**但训读会连浊：**

- 草（くさ）+ 花（はな）
   ＝ 草花（くさばな）
- 生（い）け+ 花（はな）
   ＝ 生け花（いけばな）
- 花（はな）+ 火（ひ）
   ＝ 花火（はなび）

------

## 4. 如果第二个词内部已经有浊音了，则**不再连浊**

- 为了发音顺畅考虑，不再增加额外连浊。

**例：**
 旅行（りょこう）+ かばん
 ＝ 旅行かばん（无变化，「ば」已经是浊音）

------

## 5. 音读词中也有**例外**（高频常用、历史悠久）

- 大约10%左右的音读组合，会出现连浊。

**例：**

- 旅行（りょこう）+ 会社（かいしゃ）
   ＝ 旅行会社（りょこうがいしゃ）
- 睡眠（すいみん）+ 不足（ふそく）
   ＝ 睡眠不足（すいみんぶそく）
- 文庫（ぶんこ）+ 本（ほん）
   ＝ 文庫本（ぶんこぼん）

------

## 6. 外来词基本上**不连浊**

- 绝大多数外来词组合不连浊。
- 但如果是「已经日语化非常久的外来词」，可能会出现连浊。

**例：**

- 雨（あめ）+ かっぱ（合羽，来源葡萄牙语）
   ＝ 雨がっぱ
- 歌（うた）+ かるた（纸牌，源自葡萄牙语 carta）
   ＝ 歌がるた

------

# 练习题答案

1. 子（こ）+ 猿（さる）
    ＝ 子猿（こざる）✅（连浊，正确）
2. 子（こ）+ 羊（ひつじ）
    ＝ 子羊（こひつじ）✅（不连浊，正确，因为「ひつじ」开头不是か、さ、た、は行）

# 小补充（可选了解）

- **连浊**日语叫做「連濁（れんだく）」。
- 常见连浊变化：
  - か → が
  - さ → ざ
  - た → だ
  - は → ば（有时也会变成ぱ，叫做半浊音）
